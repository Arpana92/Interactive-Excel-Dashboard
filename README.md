# Interactive-Excel-Dashboard
Interactive Sales Dashboard built in Excel to analyze Super Store performance. Includes KPIs like total &amp; average sales, transactions, quantity, region-wise analysis, ship mode distribution, monthly trends, category and segment insights, top cities, and year-based slicers for dynamic, data-driven decision-making.
